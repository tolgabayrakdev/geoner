# Geoner Project Readme.md

## Server
#### You can do these to run on your own localhost.
! Don't forget the run postgresql on your machine.
```
- python3 -m venv venv 
- pip install requirements.txt
- cd app
- uvicorn main:app --reload(run server)
```

## Client
```
- cd client
- npm install 
- npm run dev
```
